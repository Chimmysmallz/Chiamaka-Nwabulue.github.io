## Tech

Built as a **single self-contained HTML file** — all CSS, JavaScript, and images are
inline, so it runs anywhere with no build step, no frameworks, and no external requests.

## View locally

Just open `index.html` in any web browser — that's it.

## Deploy

This site works on any static host. Popular free options:

- **GitHub Pages** — Settings → Pages → deploy from the `main` branch
- **Cloudflare Pages** — direct upload
- **Tiiny.host** — drag-and-drop a single HTML file

## Edit

All content lives in `index.html`. Open it in any text editor (VS Code recommended),
use Find & Replace to update text, save, and refresh your browser to preview.
